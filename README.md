# SystemMonitor

SystemMonitor 是一個用於監控系統資源使用率的應用程式，可以即時顯示 CPU、磁碟和記憶體的使用情況。

## 系統需求

- Node.js 20 或更高版本
- pnpm 9 或更高版本

## 安裝步驟

1. 確保已安裝 Node.js 18.0.0 或更高版本
2. 安裝 pnpm 9.0.0：
   ```bash
   npm install -g pnpm@9
   ```
3. 安裝專案依賴：
   ```bash
   pnpm install
   ```

## 功能特點

- 即時監控 CPU 使用率
- 顯示磁碟使用情況
- 監控記憶體使用率
- 直觀的圖形化介面

## 開發

```bash
# 啟動開發伺服器
pnpm dev

# 建置專案
pnpm build
```
